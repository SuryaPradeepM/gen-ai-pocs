# Mini Gen-AI Prototypes

Gen AI PoCs is a collection of Proof of Concept (PoC) projects demonstrating various generative AI applications. The repository contains multiple independent projects showcasing different use cases and implementations of AI technologies, built with Python and JavaScript, currently in active development (WIP - Work In Progress).

## Mini Apps

- **AI_Presenter**: AI-powered presentation generation tool
- **HR_Chatbot**: Human Resources chatbot for employee queries
- **Logo_Generator/Logo_gen**: AI-based logo generation applications
- **Meeting_Minutes**: Automated meeting minutes transcription and summarization
- **RAG_Chatbot**: Retrieval-Augmented Generation chatbot implementation
- **UI**: User interface components for the PoC applications

### Technology Stack

- Python
- FastAPI
